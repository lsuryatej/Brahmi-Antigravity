# Sutr Product Images

This directory contains images for all Sutr collection products.

## Directory Structure

Each product has its own folder with 3 images:
- `image-1.jpg` - Main product image
- `image-2.jpg` - Alternate view/detail shot
- `image-3.jpg` - Lifestyle/styled shot

## Image Guidelines

- **Format**: JPG or PNG
- **Size**: Minimum 1200x1600px for best quality
- **Aspect Ratio**: 3:4 (portrait) recommended
- **File Size**: Keep under 500KB each (optimize for web)
- **Naming**: Must match exactly: `image-1.jpg`, `image-2.jpg`, `image-3.jpg`
