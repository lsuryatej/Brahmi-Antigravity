# Collections Images

Place your collection images here with the following names:

## Required Images:
- `ceramics.jpg` or `ceramics.png` - Ceramics collection image
- `textiles.jpg` or `textiles.png` - Textiles collection image
- `woodwork.jpg` or `woodwork.png` - Woodwork collection image
- `metal-art.jpg` or `metal-art.png` - Metal Art collection image
- `paintings.jpg` or `paintings.png` - Paintings collection image

## Image Guidelines:
- **Recommended size**: 800x1000px (portrait orientation works best)
- **Format**: JPG or PNG
- **Optimization**: Keep file sizes under 500KB for faster loading
- **Aspect ratio**: 4:5 or similar portrait ratio for best display in the carousel
